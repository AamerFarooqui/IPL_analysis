# 🏏 IPL 2022 Data Analysis | Capstone Project

## 📌 Project Overview

This project analyzes **IPL 2022 match data** to uncover insights about team performance, match outcomes, venues, and player contributions. The goal of this project is to use **Python data analysis techniques** to explore patterns in the tournament and answer key cricket-related business questions.

The project demonstrates **data cleaning, exploratory data analysis (EDA), and insight generation** using Python.

---

# 🎯 Project Objectives

* Analyze **IPL 2022 match data**
* Identify **top-performing players**
* Understand **team performance trends**
* Analyze **venue-based match outcomes**
* Extract insights from match statistics

---

# 📂 Dataset Description

The dataset contains match-level information from IPL 2022.

### Important Columns Used

| Column              | Description                         |
| ------------------- | ----------------------------------- |
| date                | Match date                          |
| venue               | Stadium where match was played      |
| stage               | Match stage (league, playoffs etc.) |
| team1               | First team                          |
| team2               | Second team                         |
| toss_winner         | Team that won the toss              |
| toss_decision       | Decision after winning toss         |
| first_ings_score    | First innings score                 |
| second_ings_score   | Second innings score                |
| match_winner        | Match winning team                  |
| won_by              | Win type (runs/wickets)             |
| margin              | Winning margin                      |
| player_of_the_match | Best player of match                |
| top_scorer          | Highest scorer in match             |
| highscore           | Highest runs scored                 |
| best_bowling        | Best bowler                         |
| best_bowling_figure | Best bowling performance            |

# 💡 Insights Generated

Some insights discovered during the analysis:

* Certain venues consistently produced **higher scoring matches**.
* Winning the **toss sometimes influenced match strategy**.
* A few players dominated **Player of the Match awards**.
* Teams chasing targets had competitive success depending on venue conditions.
